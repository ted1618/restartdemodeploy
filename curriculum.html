Become a Web Developer

HTML, PHP, MEAN.js, Full Stack Developer... If all these names ring no bell, there is no need to worry. Over 60% of our students have never coded before starting the bootcamp, and felt the same way. No prior programming knowledge is required to be successful.
Here's a simple breakdown of what you will learn.
Front End

The front end is what the users of a website see, for instance the front end of this website is displayed on your screen right now. We built this using HTML, CSS (and a tool called Bootstrap), and JavaScript (with a tool called jQuery).

Back End

The back end is the "brain" of a web application, it contains all the logic and rules the website must follow to work properly. You can write this logic in server-side languages such as PHP, and even JavaScript.

Database

Sometimes we will build applications that store information in a database, such as products in a webshop, or users on Facebook. We will learn about database design and the SQL language, and dive into NoSQL databases with MongoDB.

Learn by doing

Don't study for a job, learn because you want to leave your mark in the world.
Industry Driven Content

We work with tech companies and academics to continuously improve the curriculum and make sure it incorporates the major trends relevant in the industry today.

People Before Coders

No matter the company or industry, you will work with, and at times lead, other people. At Restart One we incorporated tracks to work on your communication, presentation and leadership skills.

An Extremely Intense Experience - In The Best Possible Way!

{ 3 months }
Bootcamp
  • An intensive (and really fun!) experience, with the aim of bringing the best out of each student.
  • Master Classes from the world's biggest tech companies, university professors, and iconic tech and business guests.
  • Front End Technologies: HTML, CSS and Bootstrap, JavaScript and jQuery, Ajax and APIs
  • Back End Technologies: SQL, PHP, OOP, PHP MVC.
  • Version Control, Web App Deployment, Cyber Security and Agile Development.
{ 3 months }
Guide
  • Succesful graduates become Guides, a period in the program where personal leadership and teamwork are required for success.
  • Guides take ownership over their technical growth by learning a new technology stack - MEAN.JS.
  • Guides take leadership roles in the bootcamp, host algorithm classes, explain concepts and manage the projects and progress of the bootcampers.
  • Advanced hands-on workshops on Data Structures and Algorithms, Security, SEO and Optimization, are organized regularly for our Guides.
{ 6 months }
Developer
  • Successful Guides continue their journey into Computer Science in the Developer stage where each one of the students is assigned to an awesome tech company in the region, through our network. All Restart One Partners are committed to support our mission and the growth of our students.
  • For 6 months students continue developing their skills by being immersed in a real-world development environment, working with seasoned developers.
  • At the end of Restart One all students have a strong technical portfolio and a personal network to find their first full time position.
